# 🛒 POS Machine Project

A **Point of Sale (POS) System** developed in **C++** using **Object-Oriented Programming (OOP)** principles. This project was created as part of our university OOP course to simulate a retail management system that simplifies daily shop operations, including sales processing, inventory management, stock updates, and receipt generation.

---

## 📌 Overview

The POS Machine Project is a console-based application designed to help shopkeepers manage products and sales efficiently. It provides a simple and user-friendly interface for handling day-to-day retail operations while demonstrating core Object-Oriented Programming concepts in C++.

---

## ✨ Features

- 🛍️ Make Sales
- 📦 View Inventory
- ➕ Add New Stock
- 🔄 Update Existing Stock
- 🧾 Generate Customer Receipts
- 💾 Maintain Product Records
- 🚪 Secure System Exit
